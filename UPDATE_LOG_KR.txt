[ Video To GIF Tool - v2.0 업데이트 노트 ]
날짜: 2025-12-11

이번 업데이트는 기존 Tkinter 기반의 한계를 극복하고, 성능과 사용자 편의성을 대폭 강화하기 위해 엔진을 PyQt6로 전면 교체한 메이저 업데이트입니다.

--------------------------------------------------------------------------------
1. 핵심 변경 사항 (Core Changes)

■ 엔진 교체 (Tkinter → PyQt6)
 - GPU 가속을 지원하는 QMediaPlayer 도입으로 고화질 영상도 끊김 없이 미리보기 가능.
 - UI 반응 속도 개선 및 창 이동 시 멈춤 현상 해결.
 - 세련된 다크 테마(Dark Theme) 스타일시트 적용.

■ 용량 예측 기능 강화 (Smart Estimation)
 - 구버전의 강력했던 '2.0초 정밀 샘플링' 로직을 복원하여 예측 정확도 향상.
 - [New] 일괄 예측(Batch Estimation): 목록에 있는 여러 파일의 예상 용량을 한 번에 계산하여 팝업으로 제공.

■ 변환 엔진 최적화
 - FFmpeg + Gifski 파이프라인 최적화 (yuv4mpegpipe 사용).
 - 변환 중 '취소(Cancel)' 기능 추가.
 - WebP 포맷 지원 강화.

--------------------------------------------------------------------------------
2. 사용자 편의성 (Usability)

■ 설정 저장 (Persistence)
 - 언어 설정(한글/영어) 및 사용자 환경 설정이 프로그램 종료 후에도 자동 저장/유지됨.

■ 비주얼 타임라인 (Visual Timeline)
 - 구간 설정 시 시작/종료 지점을 슬라이더 상에 시각적으로 표시.
 - 드래그 앤 드롭으로 더욱 직관적인 구간 조절 가능.

■ 일괄 처리 (Batch Processing)
 - 다수 파일 드래그 앤 드롭 지원.
 - 파일별 개별 설정(FPS, 해상도 등) 적용 가능.
 - 선택된 파일만 골라서 변환하거나 예측 가능.

--------------------------------------------------------------------------------
3. 기타 개선 사항 (Improvements)

 - 상태 표시줄(Status Bar) 추가로 작업 진행 상황 실시간 확인.
 - FPS 자동 감지 로직 개선 (OpenCV 활용).
 - 아이콘(app_icon.ico) 및 필수 리소스(ffmpeg/gifski) 내장 빌드 지원 (포터블 실행 파일).
 - UI 레이아웃 직관성 개선 (버튼 배치 최적화, 폰트 가독성 향상).
