[ 움짤 메이커 (GifClip Maker - v3.0) ]

Python(PyQt6)을 기반으로 제작된 올인원 고품질 동영상 GIF/WebP 변환 도구입니다.
세련된 다크 테마 UI와 강력한 편집 기능(자르기, 크롭, 리사이즈)을 통해 누구나 쉽게 고품질 '움짤'을 만들 수 있습니다.

FFmpeg의 강력한 인코딩 성능과 Gifski의 고화질 GIF 압축 엔진을 결합하여 최고의 결과물을 만들어냅니다.

--------------------------------------------------------------------------------
[ v3.0 주요 업데이트 기능 ]

1. 강력한 영상 편집 도구
   - 크롭(Crop): 화면의 특정 부분만 잘라낼 수 있는 직관적인 오버레이 UI를 제공합니다.
     * 프리셋 지원: 1:1, 16:9, 9:16, 4:3, 자유 비율
     * 마우스로 드래그하여 영역을 직접 조절할 수 있습니다.
   - 정밀한 리사이즈: 
     * 원본 유지, 비율 축소(75%, 50%, 25%), 사용자 지정 해상도 등 다양한 옵션을 제공합니다.
   - 키보드 제어: 스페이스바(SPACE)로 동영상을 재생/일시정지 할 수 있습니다.

2. 사용자 편의성 대폭 개선
   - 메뉴 바 도입: 파일 열기, 언어 설정 등이 상단 메뉴로 깔끔하게 정리되어 작업 공간이 넓어졌습니다.
   - 개선된 UI 레이아웃: 일관된 여백과 정렬로 더욱 전문적이고 깔끔한 외관을 제공합니다.
   - 비디오 로드시 첫 화면이 검은색이 아닌 비디오 첫 프레임이 바로 보이도록 수정

3. 버그 수정
   - WebP 컨버트시 용량이 비정상적으로 커지던 문제 수정

4. 검증된 성능
   - 고화질 변환: Gifski 엔진을 사용하여 색상 손실을 최소화한 최고급 움짤을 생성합니다.
   - 용량 예측: 변환 전 예상 파일 크기를 미리 계산해 볼 수 있습니다. (2초 샘플링 방식)
   - 멀티스레드: 변환 중에도 UI가 멈추지 않으며, 안정적으로 작업이 진행됩니다.

--------------------------------------------------------------------------------
[ 상세 기능 목록 ]

1. 형식(Format)
   - GIF: 호환성이 뛰어난 움직이는 이미지.
   - WebP: GIF 대비 더 높은 압축률과 화질을 제공하는 차세대 포맷.

2. 비디오 제어
   - 구간 자르기: 타임라인 슬라이더를 드래그하거나 [시작/종료 설정] 버튼으로 원하는 움짤 구간만 추출.
   - 재생/일시정지: 스페이스바 또는 버튼 클릭.
   - 타임라인 이동: 슬라이더를 통해 원하는 프레임으로 즉시 이동.

3. 품질 및 크기 제어
   - FPS (프레임 등급): 부드러운 움짤을 위한 초당 프레임 설정.
   - 품질(Quality): 용량과 화질 사이의 밸런스 조절 (1~100).
   - 해상도: FHD, HD 등 프리셋 또는 직접 입력 리사이즈.

4. 일괄 처리
   - 여러 비디오 파일을 한 번에 등록하여 연속으로 관리할 수 있습니다.
   - 파일마다 각기 다른 설정을 저장하여 개별적으로 변환하거나 일괄 변환이 가능합니다.

--------------------------------------------------------------------------------
[ 사용 방법 ]

1. [파일] 메뉴 -> [비디오 열기]를 선택하거나, 동영상을 프로그램 창으로 드래그합니다.
2. 하단의 노란색 타임라인 슬라이더를 조절하거나 [시작 설정]/[종료 설정] 버튼으로 움짤로 만들 구간을 선택합니다.
3. (선택 사항) [Crop] 체크박스를 켜서 화면의 일부분만 잘라냅니다.
4. 우측 설정창에서 출력 포맷, FPS, 품질, 크기를 설정합니다.
5. 리스트에서 파일을 선택하고 [용량 예측]을 눌러보거나, 바로 [변환하기] 버튼을 누릅니다.
6. 변환이 완료되면 원본 파일이 있는 폴더에 결과물이 저장됩니다.
