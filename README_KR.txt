[ Video To GIF Tool (v2.0 - PyQt6 Edition) ]

Python(PyQt6)으로 제작된 강력한 동영상 GIF/WebP 변환 도구입니다.
현대적인 다크 테마 UI를 제공하며, FFmpeg와 Gifski를 사용하여 최고의 변환 화질과 성능을 자랑합니다.

--------------------------------------------------------------------------------
[ 주요 기능 ]

1. 핵심 기능
   - 고품질 변환: Gifski 엔진을 사용하여 최적의 GIF 화질(색상/디더링)을 제공합니다.
   - 포맷 지원: GIF 및 WebP 움짤 생성을 지원합니다.
   - 일괄 처리 (Batch):
     * 여러 동영상을 목록에 추가하여 관리할 수 있습니다.
     * 각 동영상마다 개별 설정(FPS, 크기, 품질)을 적용할 수 있습니다.
     * 선택한 항목만 변환하거나 전체를 일괄 변환할 수 있습니다.
   - 정확한 용량 예측: 2.0초 샘플링 방식을 사용하여 결과물의 용량을 정확히 미리 예측합니다.
   - 설정 자동 저장: 언어 설정 및 사용자가 변경한 옵션이 자동 저장됩니다.

2. 최신 UI 및 사용자 경험
   - 부드러운 미리보기: GPU 가속(QMediaPlayer)을 사용하여 끊김 없는 실시간 재생을 지원합니다.
   - 비주얼 타임라인: 드래그 앤 드롭 슬라이더로 시작/종료 구간을 직관적으로 설정합니다.
   - 드래그 앤 드롭: 동영상 파일을 창으로 끌어다 놓으면 바로 로드됩니다.
   - 다크 테마: 눈이 편안하고 세련된 다크 모드 인터페이스를 제공합니다.

3. 정밀 제어
   - 구간 설정: 슬라이더로 시각적으로 설정하거나 직접 시간(mm:ss)을 입력할 수 있습니다.
   - 해상도 조절: 프리셋(1080p, 720p), 사용자 지정(폭x높이), 비율(%) 등 다양한 옵션 제공.
   - FPS 및 품질: 원본 FPS 자동 감지 및 1-100 단계의 품질 조절 슬라이더.
   - 다국어 지원: 한국어 및 영어(English)를 완벽 지원합니다.

--------------------------------------------------------------------------------
[ 사용 방법 ]

1. 실행: VideoToGifTool.exe 파일을 실행합니다.
2. 파일 추가: 동영상을 창으로 드래그하거나 [비디오 열기] 버튼을 누릅니다.
3. 구간 선택: 하단 타임라인 슬라이더를 조절하거나 [시작/종료 설정] 버튼으로 구간을 자릅니다.
4. 설정: 우측 패널에서 FPS, 크기, 품질을 원하는 대로 조절합니다.
5. 용량 예측 (선택 사항): 목록에서 파일을 선택하고 [용량 예측] 버튼을 눌러 결과 크기를 미리 확인합니다.
6. 변환: [변환하기] 버튼을 누르면 작업이 시작됩니다.
   * 결과물은 원본 동영상이 있는 폴더에 저장됩니다.

--------------------------------------------------------------------------------
[ 시스템 요구사항 ]

- Windows 10 / 11 (64비트 권장)
- 별도 설치 불필요 (포터블 실행 파일)
- ffmpeg.exe 및 gifski.exe는 내장되어 있어 별도 다운로드가 필요 없습니다.
