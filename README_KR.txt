VtoG Tool
비디오 파일을 GIF로 변환하기 위한 간단한 GUI 툴입니다.
이 프로그램은 Python(tkinter) 기반으로 제작되었으며 ffmpeg와 gifski를 사용해 고품질 GIF 파일을 생성합니다.

기능 소개
비디오 파일을 GIF 형식으로 변환 (mp4, mkv, avi, mov, webm, wmv 지원)
타임라인 슬라이더를 이용한 실시간 비디오 미리보기
프로그램 내에서 비디오 재생 및 일시정지 가능
현재 프레임 기준으로 시작 구간(Start) 설정 가능
현재 프레임 기준으로 종료 구간(End) 설정 가능
초 또는 시간 형식(mm:ss, hh:mm:ss)으로 구간 직접 입력 가능
선택한 변환 구간을 타임라인 바에서 시각적으로 표시
GIF 품질을 1~100 범위로 설정 가능 (gifski 기반)
출력 GIF의 FPS 값을 직접 설정 가능
원본 해상도 유지 옵션 제공
미리 정의된 해상도 프리셋 선택 가능 (1080p, 720p 등)
사용자 지정 픽셀 해상도(가로/세로) 설정 가능
원본 대비 퍼센트 기반 해상도 스케일 조절 가능
변환 전에 GIF 예상 용량을 미리 확인 가능
드래그 앤 드롭으로 비디오 파일 바로 열기 가능 (선택 기능)
변환 중 CMD 창이 표시되지 않음
프로그램 하단 상태 메시지로 진행 상황 표시
변환된 GIF 파일은 원본 비디오와 동일한 폴더에 저장됨

사용 방법
압축 파일을 하나의 폴더에 모두 해제
프로그램 실행 파일 실행
Open Video 버튼을 눌러 비디오 파일 선택 또는 파일을 창으로 드래그 앤 드롭
타임라인 슬라이더로 영상 확인
필요 시 시작 및 종료 구간 설정
품질, FPS, 해상도 옵션 설정
필요 시 Estimate Size 버튼으로 용량 예상
Convert to GIF 버튼 클릭
원본 비디오와 동일한 위치에 GIF 파일 생성

동봉 파일
VtoG 실행 파일
ffmpeg.exe (공식 빌드)
gifski.exe (공식 빌드)

필수 환경
Windows 운영체제
별도의 추가 설치 필요 없음
ffmpeg 및 gifski 실행 파일은 이미 포함되어 있음

참고 사항
ffmpeg.exe 및 gifski.exe 파일은 반드시 프로그램과 같은 폴더에 있어야 합니다
출력 해상도는 호환성을 위해 자동으로 짝수 값으로 조정됩니다
매우 짧은 영상의 경우 용량 예상 정확도가 떨어질 수 있습니다
변환 작업은 백그라운드에서 실행되어 UI가 멈추지 않습니다

라이선스
이 프로그램은 무료로 제공됩니다
ffmpeg 및 gifski는 수정되지 않은 공식 바이너리로 동봉됩니다
각 툴의 라이선스 정책을 따릅니다
